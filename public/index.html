API Hari Libur Nasional

Cara Mengakses API

  • /api untuk mendapatkan daftar hari libur tahun sekarang.
  • /api?bulan=2&tahun=2023 gunakan parameter bulan dan tahun untuk mendapatkan daftar hari libur di bulan 2 tahun 2023.
  • /api?tahun=2022 gunakan parameter tahun tuntuk mendapatkan daftat hari libur di tahun yang di inginkan.
  • /api?bulan=8 gunakan parameter bulan untuk mendapatkan daftar hari libur di bulan 2 tahun sekarang.
  • /api/next untuk mendapatkan hari libur berikutnya.

Apabila data tidak ditemukan respon akan berupa array kosong

Bagaimana cara kerja API ini?

API ini mengambil data dari tanggalan.com dan data per tahun ditarik setiap satu bulan sekali menggunakan GitHub Actions.

Apabila mencoba mengakses data yang belum ditarik, server akan melakukan scraping ke web target
